# Multimodal Product Review Sentiment + Helpfulness Ranker

This project builds a multimodal machine learning system that analyses product reviews using both text and images.

## Objectives
- Predict sentiment of product reviews
- Rank review helpfulness
- Combine text and image features

## Models
- BERT for sentiment analysis
- CLIP for image-text embeddings
- Late fusion for multimodal representation
- LambdaRank for helpfulness ranking

## Tech Stack
Python, PyTorch, HuggingFace Transformers, CLIP, FastAPI, MLflow, DVC, Docker, GitHub Actions.
